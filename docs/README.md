# Documentation site

- Static showcase for the new components.
- How to view locally and on GitHub Pages.

Viewing locally:
- Serve the docs directory with any static server, e.g. `npx http-server docs -p 4000`.
- Open http://localhost:4000/.

GitHub Pages:
- Enable Pages to serve from the main branch's /docs (docs/ directory).
- Your site will be available at https://<user>.github.io/<repo>/.
